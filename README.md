# BIYORA – Premium African Luxury E-commerce

A modern, elegant e-commerce platform showcasing premium handcrafted African textiles and luxury fashion.

![BIYORA Banner](https://via.placeholder.com/1200x400/0a0a0a/d4af37?text=BIYORA+-+Timeless+African+Luxury)

## ✨ Features

- **Modern Tech Stack**: Next.js 15 (App Router), React 19, TypeScript, Tailwind CSS
- **Smart Cart System**: Zustand state management with `localStorage` persistence
- **Advanced Filtering**: Real-time search + category filters
- **Beautiful UI**: Luxury dark theme with gold accents
- **Product Experience**: Clickable cards with detailed modal
- **Secure Payment**: Paystack integration (Nigeria’s leading payment gateway)
- **Order Success Page**: Clean confirmation with reference number
